ShadowWalker

Role: Game Programmer (Solo) · Engine: Unity

A maze-based horror escape game where light damages the player and darkness heals them — a unique inversion of survival mechanics that creates intense tension.

Key Features

  • ⚡ Light vs Dark mechanic — light damages, darkness heals
  • 🤖 Enemy AI with NPC chase & patrol system
  • 🌀 Dynamic maze with disappearing/shifting walls
  • ⚙️ Difficulty system — adaptive challenge scaling

⚠️ Challenge

Balancing survival mechanics — ensuring light/dark gameplay felt fair yet tense.

✅ Solution

Adjusted healing/damage values per difficulty and tuned AI behavior for better gameplay balance through extensive playtesting.

Zombie Shooter

Role: Game Programmer (Solo) · Engine: Unity

A wave-based zombie survival game where players must fight off increasingly dangerous hordes and strategically progress through levels with limited resources.

Key Features

  • 🌊 Wave-based zombie spawning system
  • 🧠 Enemy AI follow & aggro system
  • ☠️ Poison attack mechanics
  • 🗝️ Key collection & progression system

⚠️ Challenge

Managing increasing wave difficulty while keeping gameplay accessible for new players.

✅ Solution

Implemented progressive wave scaling with spawn rate curves and enemy health multipliers tied to wave number.
