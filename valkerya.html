A UE5 Fantasy Action-Adventure

VALKERYA

VALKERYA is a third-person fantasy action-adventure game set in a world consumed by darkness and ancient corruption. Players explore handcrafted environments, solve environmental puzzles, battle corrupted enemies, and uncover the forgotten history of the kingdom through cinematic storytelling and immersive gameplay.

View Project Details
// Overview

About the Project

VALKERYA is a collaborative university game development project created in Unreal Engine 5. The game combines exploration, combat, environmental storytelling, and cinematic presentation to create an immersive fantasy adventure.

Our team's focus was on building engaging gameplay, intuitive user interfaces, memorable environments, and a polished player experience that holds up to modern AAA standards.

Project Type University Project
Team Structure 5 Developers
Engine Unreal Engine 5
Genre Fantasy Action-Adventure
Duration 4 Months
Platform PC (Steam)

// Debugging

Challenges & Solutions

⚠️ Balancing Mechanics

Coordinating gameplay combat loops and ensuring checkpoints felt rewarding yet challenging for players of different skill levels was difficult. Solutions involved implementing progressive stat scaling, setting curves for enemies, and structuring early playtest groups.

✅ Engine & Asset Integration

Combining visual elements built across Photoshop, Blender, and integrating custom UI assets into Unreal Engine and Unity while maintaining strict texture performance parameters. Solved through proper sprite packing, mip-map optimization, and clean asset organization.
